# Reunion Report: Reference Extraction Skill
**Date**: 2026-04-24
**Context**: [REUNION]

## Problems Identified
- Agents having problems with targeted reading of `architecture.md`.

- Updated `agent_rules.md`:
    - Refined Rule **1.3 Targeted Extraction** to mandate the use of the skill.
    - Added Section **2. Agent Skills (SKL)** to document the `RefExtractor` skill.
    - Renamed following sections to maintain numbering.

## Files Modified

- `agent_rules.md`
