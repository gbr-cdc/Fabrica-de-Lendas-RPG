## Changes Made

### `agent_rules.md`
- Removed Rule 1.3 (File Closure) — delegated entirely to `[EXECUTION]` context.
- Added Rule 1.3 **Reference Extraction**: mandates targeted extraction (grep/view_file ranges) instead of full file reads for `architecture.md` and `MISSION_HISTORY.md`.

### `docs/contexts/execution.md`
- Renamed **Mandatory Reading** → **Mandatory Extraction** with explicit "ONLY / Targeted" qualifiers per file.
- Expanded **StepComplete** (Step 4) to include **Context Cleanup**: close files not in scope of the next step; closing strips file content from memory and the step note becomes its memory replacement.
- Merged former Steps 5 & 6 into a single **MissionComplete** gate (Archive → Sync), preventing premature success reporting.

### `docs/contexts/reunion.md`
- Replaced Step 4 (Commit) with **Report & Commit** phase: close file contexts → generate reunion report from conversation memory → remind USER to commit manually with suggested message.
