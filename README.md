# Fabrica-de-Lendas-RPG

Este projeto é um sistema de RPG que estou desenvolvendo que tem como foco cenários de fantasia e no estilo anime de batalha. Ele tem o diferencial de implementar um controle de turnos de combate que busca emular o ATB (Active Time Battle) de jogos como Final Fantasy IV e Chrono Trigger, de forma que personagens mais rápidos agem com mais frequência do que personagens mais lentos, mas de maneira equilibrada que não quebre a economia de ações e faça com que personagens lentos não consigam jogar. Além disso, o sistema busca ser bastante simples e fluido tanto nas suas resoluções de combate quanto em testes de perícias e habilidades para a hora da interpretação.

Fora isso, este projeto originou a criação de um script em Python para fazer testes de balanceamento entre as classes. Isto acabou por se expandir em uma engine de combate que usa padrões de projeto sofisticados como o *Observer Pattern* e o *Command Pattern* para implementar um gerenciador de batalha que consiga lidar com as lógicas das habilidades, e uma *Min-Heap* para controlar a ordem de turnos de maneira eficiente. O projeto utiliza OO (Orientação a Objetos) e está bem organizado e modularizado com aplicação de conceitos como herança e polimorfismo e um bom gerenciamento das responsabilidades das classes, de forma que ele pode depois ser portado para uma engine como Godot ou Unity para a criação de um jogo que utilize o sistema Fábrica de Lendas como regra.
